Interactive geospatial visualization of Indian states and Union Territories with capitals, featuring custom styling, hover tooltips, and dynamic highlighting using Python (Folium).
# 🇮🇳 India Geospatial Visualization

An interactive map visualizing all 28 Indian states and 8 Union Territories along with their respective capitals.

## Features
- Hover tooltips displaying state name and capital
- Dynamic highlight effect on interaction
- Custom dark theme with teal-to-gold color palette
- Smooth hover animations
- Complete coverage of Indian administrative regions

## Tech Stack
- Python
- Folium
- GeoJSON
- Google Colab

## Future Improvements
- Add population or demographic data overlays
- Enable filtering by region
- Integrate real-time data sources
